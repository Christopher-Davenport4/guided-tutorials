# Guided Tutorials

Tutorial-driven builds I completed to learn BI and analytics tools. These are
follow-alongs: the design decisions belong to the original authors, and I built
them to get hands-on with the tooling, not to showcase my own analytical or
design judgment. My independent, from-scratch work lives in separate repos.

Each project folder records the source I followed and what the build taught me.

## Structure

Organized by tool, then by project:

- `power-bi/`
  - `data-professional-survey-breakdown/` — dashboard build covering cards,
    treemap, stacked bars, gauges, and a donut across a data-professional survey.

## Note

Because these are guided, some design choices reflect the tutorial rather than
what I would choose independently. Where a build taught me something I would now
do differently, I note it in that project's README.
